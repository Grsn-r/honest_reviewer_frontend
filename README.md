Proyecto: honest reviewer
autor: Gerson Ruíz sequeda

Manejo de registro de usuarios verificando que el correo no esté duplicado, de ser así el usuairo recibe un aviso
El usuario es redirigido a login con un registro exitoso
El usuario recibe un aviso si la contraseña es errónea
La página principal está protegida, solo se accede con cuenta
Se pueden modificar datos de perfil y la contraseña
Cambio de contraseña requiere verificación
Puede publicar, comentar y reaccionar
Puede eliminar su contenido pero no el de otros incluyendo los comentarios
Diseño responsivo para teléfonos, tablets y laptops
autenticación de usuario via token
Se pueden subir imágenes
